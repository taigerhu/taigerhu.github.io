我的PC组装升级记

2024-05-18

18年左右为了编译Android组装了一台PC，AMD AM4，2K，主要配置如下：

  • 主板：技嘉 GA-A320M-S2H (rev. 1.x)
  • CPU ：AMD Ryzen 3 2200G (盒装) 1100 (+mb)
  • 内存：威刚 DDR4 2400MHz 8G 439
  • 硬盘：金泰克 SSD 480G 400
  • 主机：大水牛/300W 200

最近试试编译OH4.0花了6h40m：

这速度，能急死，升级下了，一开始只是想升级下CPU，到后面越升越多：）

先看了下主板CPU support列表, 好家伙AM4支持的是真多，2200G:

AMD Ryzen 3 2200G 4C/4T 3.5GHz / 3.7GHz 2MB 4MB 1100 MHz Raven Ridge 14nm B0 45-65W F10

R3 2200G只有4个核，选个6个的吧，太高了怕带不动。

最后看了BIOS版本大多到F55支持的比较多。因为用来编译做server用，集显就不是必选了，而且比非集显相对也较贵, 而且这主板版本供电貌似稍弱，最后看上了R5 5500：

AMD Ryzen 5 5500 6C/12T 3.6GHz / 4.2GHz 3MB 16MB N/A Cezanne 7nm A0 65W F55

7nm工艺，jd自yin盒装559，jdcard还有100，449拿下，后面在xianyu把旧的还卖了220, 也就是花了200+升级了CPU，哈～双。

下来就是买个显卡，能点亮就行了, 也就装机or偶尔用下，tao个二手便宜的吧, 不需要额外供电那种，一开始买的昂达R5 220 1GB，100，但是大晚上的能听到zizi声音, 强迫症，受不了退了，换了个七彩虹GT710 1G, 这个帅，静音，110。

内存看OH官方推荐16GB+：

Ubuntu环境要求

Ubuntu18.04及以上版本，X86_64架构，内存推荐16 GB及以上。

Ubuntu系统的用户名不能包含中文字符。

我这8G也升下吧，而且是2400的频，看了看这个主板内存说明:

1.2个DDR4 DIMM插槽，可支持高达32 GB
2.支持双通道内存技术
3.支持 DDR4 3200(O.C.)/2933(O.C.)/2667*/2400/2133 MHz
* 是否支持此规格依CPU而定。
4.支持ECC Un-buffered DIMM 1Rx8/2Rx8内存(non-ECC模式运行)
5.支持non-ECC Un-buffered DIMM 1Rx8/2Rx8/1Rx16内存
6.支持Extreme Memory Profile (XMP)内存

最大支持3200, 不过要O.C.(OverClocking), 看了看主流条子说要主板BIOS开启XMP，我这主板支持阿，大不了退到2400，搞一个试试，考虑到兼容性还是买了威刚，万紫千红版，晒单返现249 - 10 = 239拿下，回来插上去直接上了3200，哈～，都不用动BIOS, sudo dmidecode --type 17如：

这个大水牛机箱外观还凑合，电源上置。偶尔搜到一个帖子，发现Tt启航者A1颇具美感（尤其是外露下置电源 BT 500W），侧拉钢化玻璃，199，不过F1亚克力侧版149更具性价比，据说钢化玻璃侧板固定要像A1这样才安全些，所以类似平头哥的款式就没选了，而且启航者这名字听上去感觉也更好点，哈哈。

原来的电源外观是真不行，搭这个机箱不太配，BT 500W官方已经不生产了。。。不过，竟给我发现一个官方好物：Tt Smart G 550W 金牌/全模组在售，优惠价只要249，赚了哈感觉。

这个主板还支持M.2 SSD:

1个M.2插座 (支持Socket 3，M key，type 2242/2260/2280/22110 SATA 及 PCIe x4/x2* SSD)

5.1 闲着没事干，看中了七彩虹 CN600 PRO M.2(NVMe)，就装个Ubuntu，code放原来固态上，挂到/home，256G够了 149，不错。颗粒好像是国产，不过七彩虹也算是个品牌了，图个便宜，坏了再换个就是了，就不需要swap分区了。

算下来花了大概1000的样子。

一开始只换的CPU，起不来，后来把显卡接上，提示如下：

原来是检测到new cpu，让我选Y or N，先N不过，后Y pass了。

系统Ubuntu 18.04原先一直装不上，最后装的是18.08，现在官方OH也支持20.04，就试了一把，我去，20.04可以装拉，好好。

最后只需要2h30m，速度提升60%，哈～
